Variables y tipos de Datos

Una de las cosas más llamativas de python es que el declarar una variable no necesitas especificar el tipo de dato y esto hace llamativo

Introducción:

En python declarar variables es la cosa más sencilla del mundo solo pones el nombre, igual y el valor.

Al no necesitar declarar de manera explicita el tipo de dato a las varibles, al ingresar el valor python reconoce el tipo de acuerdo a como lo ingreses ej:

  • nombre="Karen" <--- las comillas le indican que es String
  • edad=22 <--- numeros directamente sin comillas son del tipo mumerico tal como int o float
  • verdad=True <--- True o False para boolean
  • nombres=["Karen","Jorge","Israel"] <--- Los corchetes indican array

Para este ejercicio usted tambien debera saber ciertas cosas como print("hola") para mostrar en consola cualquier cosa e if(condicion):

Declara una variable llamada año_nacimiento y otra llamada año_actual en donde almacenaras los años respectivos, luego declara una variable llamada edad en donde calcularas la edad con las dos variables previas, luego si su edad es mayor igual a 18 mostrar un mensaje que diga usted es mayor de edad.


Estructuras de Control condicionales

En python las estructuras de control funcionan de manera muy parecida a la mayoria de lenguaje, con la unica diferencia de que se usan dos puntos en vez de llaves y su cuerpo esta definido con un espaciado diferente al resto.

Introducción:

La manera de usar las sentencias de control es con la palabra reservaba y la condicion

Las sentencias existentes son

  • if(condicion):
  • if(condicion):...else:
  • if(condicion):...elfi(condicion):
